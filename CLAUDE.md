# DC Nail Supply LLC App — Claude Instructions

## Project Goal
Build an iOS-first nail supply e-commerce mobile app based on the uploaded DC Nail Supply mockup.

## Tech Stack
- Expo React Native
- TypeScript
- Local mock data first
- Expo Router will be added after baseline is stable
- No backend, payment, or real auth until UI prototype is stable

## Working Rules
- Always plan before editing.
- Do not edit files until the user approves the plan.
- Work in small milestones.
- Keep UI close to the mockup.
- Use clean TypeScript.
- Avoid adding unnecessary libraries.
- Do not implement backend/payment/auth unless explicitly requested.
- Do not auto-commit unless explicitly requested.
